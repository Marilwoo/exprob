This node is the oracle.

It gives hint and checks the winning hypothesis

None

Description:
This node is the one that manages the hints that the robot asks. When the robot asks for a hint using the request of the service of type hint, the oracle answers by sending a random hint. For doing that it takes randomly one of the possible hypothesis ID and then choses randomly one hint inside oif the hypothesis. Then, for being sure that the same hint is not given more that once the chosen hint is removed form the list. So, finally it sends back to the robot the hint as a string and the ID of the hypothesis as an int. This node also checks if the hypothesis is the winning one. The robot sends the hypothesis ID throug the service of type win and the oracle compares it with the winning ID. The oracle then sends a booleand back: True if they match and False if not.
